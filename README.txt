TRANG WEB XIN LOI NHO EM

1. Mở file index.html bằng Chrome, Edge hoặc Safari.
2. Trang web đã có sẵn:
   - Gấu xin lỗi dễ thương.
   - Hiệu ứng khi cuộn.
   - Nút "Còn giận xíu" chạy né vui vẻ nhưng không ép người nhận.
   - Nút "Hết giận rồi" tạo hiệu ứng tim, hoa và pháo giấy.
   - Nội dung xin lỗi vừa hài hước vừa chân thành.
   - Chữ ký: Huỳnh Hiệp.
3. Muốn đổi tên người nhận, tìm chữ "nhỏ em" trong index.html và thay bằng tên mong muốn.
4. Muốn đổi lời xin lỗi, chỉnh phần có tiêu đề "Lời xin lỗi thật lòng".
